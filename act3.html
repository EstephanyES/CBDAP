Primera forma normal



Para que una tabla este en INF, todos los atributos deben tener valores atomicos, lo que significa que cada campo debe contener un solo valor indivisible. Esto evita la duplicacion de datos y asegura que no haya atributos con multiples valores en una sola celda.

- Una Liave Foranea es un campo en una tabla que es una llave primaria en otra tabla y se usa para crear relaciones entre tablas
